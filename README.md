# Cipher

Ce programme permet de **chiffrer** et de **déchiffrer** des messages entrés par un utilisateur mais également le contenu de fichiers textes en proposant **différentes méthodes de chiffrement synchrones** comme le *Code de César* et le *Chiffre de Vigenère*.
